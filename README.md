# Soil Stability Analyzer

This project is a simple Python-based environmental data analysis tool that evaluates soil moisture levels to estimate potential stability conditions and landslide risk.

## Purpose
The goal of this project is to demonstrate how basic computational methods can be applied to environmental science problems.

## Method
- Soil moisture values are analyzed using basic statistical operations
- Average, minimum, and maximum values are calculated
- A rule-based system classifies stability risk levels

## Output Interpretation
- Low moisture → possible instability
- Moderate moisture → stable conditions
- High moisture → potential landslide risk

## Tools
- Python

## Note
This project was developed as an independent learning exercise in environmental data analysis and basic programming.
